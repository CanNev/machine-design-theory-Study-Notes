安全系数计算

  1. 根据轴的运动状态,判断轴所受弯曲应力和扭转切应力的的变化情况

    • 工作时稳定的单向旋转，轴所受的弯曲应力为 对称循环应力， 扭转切 应力为静应力
    • 工作时频繁正反转，轴所受的的弯曲应力为 对称循环应力， 扭转切应力为 对称循环应力
    • 工作时单项旋转且经常开车停车，轴所受的的弯曲应力为 对称循环应力， 扭转切应力为脉动循环应力

  2. 计算弯曲应力和扭转切应力

    • 弯曲应力
      σb = M / W
      W = πd3 / 32

    • 扭转切应力
      τ = T / WT
      WT = πd3 / 16

  3. 根据得出的应力状态的应力值计算对应的平均应力m 和应力幅a

    应力类型与参数对照表（适用σ和τ）
    应力类型 应力比r 平均应力τm 应力幅τa
    静应力 1 τ 0
    对称循环应力 -1 0 τ
    脉动循环应力 0 τ / 2 τ / 2

  4. 计算安全系数Sσ和Sτ

    弯曲应力的安全系数为
    Sσ = σ-1 / Kσσa + φσσm

    扭转应力的安全系数为
    Sτ = τ-1 / Kττa + φττm
